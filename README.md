# 🚨 Silent SOS – Emergency Alert System

## Overview

Silent SOS is a web-based emergency alert application that enables users to quickly notify a trusted contact during an emergency. With a single click, the application captures the user's current location and sends an email containing a custom emergency message along with a Google Maps link to the user's live location.

The project demonstrates how web technologies and browser APIs can be used to build practical, real-world safety solutions.

## Features

* 🚨 One-click SOS activation
* 📍 Fetches the user's current live location using the Geolocation API
* 📧 Sends an emergency email to a trusted contact
* 🗺️ Includes a Google Maps location link in the email
* 📱 Responsive and user-friendly interface

## Tech Stack

* HTML5
* CSS3
* JavaScript (ES6)
* EmailJS (or the email service you used)

## How It Works

1. User clicks the **SOS** button.
2. The browser retrieves the user's current location.
3. A Google Maps link is generated.
4. An email containing the emergency message and location is automatically sent to the registered emergency contact.
